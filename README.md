# 📊 Dashboard UI - Admin Panel Design

## 📌 Project Overview

This project is a modern **Dashboard UI design** created using Figma.
It includes multiple dashboard layouts such as E-commerce, Stock Market, and Product Management, focusing on data visualization and clean user experience.
